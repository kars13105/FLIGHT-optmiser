# ✈️ SkyRoute Optimiser

> **Find the optimal path for your next journey.**  
> **`#happiness.h is the key of happiness`**

SkyRoute Optimiser is a premium, modern web application designed to help travelers find the most efficient and cost-effective flight routes between major global destinations. With a stunning glassmorphic UI, rich interactive components, and intelligent path optimization algorithms, finding the perfect route has never been smoother.

🔗 **Live Link:** [Launch SkyRoute Optimiser](https://frontend-kappa-six-0y0l7ef9g4.vercel.app)

---

## ✨ Features

- **🔍 Smart Path Optimization**: Compare and sort flight connections by **Minimum Cost** or **Minimum Duration** with a single click.
- **🗺️ Interactive Autocomplete Airport Search**: No need to memorize complex IATA codes! Type by **city name**, **country**, **airport name**, or search by code directly using our premium suggestions dropdown.
- **📈 Visual Flight Timeline**: A responsive progress pipeline displaying your complete flight path, including origins, destinations, transit layovers, and partner airlines.
- **⚡ Dual-source API Engine**: 
  - **Live Flight Shopping**: Connects to the **Amadeus API GDS** to retrieve real-world flight options, dynamic fares, and schedules.
  - **Smart Mock Fallback**: Includes a robust built-in mock flight generator allowing zero-dependency local testing and fast offline demonstrations.
-  **Ultra-Premium Design**: Responsive dark-mode interface styled with vibrant indigo & cyan accents, backdrop blur filters, custom scrollbars, and fluid animations.
